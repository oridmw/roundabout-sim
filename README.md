# Traffic Simulator

> Harvard AM215 Final Project (Fall 2025)

To set up the project, create a virtual environment and install the required packages:

```bash
pip install -r requirements.txt
pip install -e .
```

To run the simulation (with pygame-based GUI):

```bash
python -m traffic_sim --help  # See options for tuning the simulation parameters
python -m traffic_sim
```

To run the test suite:

```bash
pytest
```
